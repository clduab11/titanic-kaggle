# Data Directory

Place your Kaggle Titanic competition data files here:

- `train.csv` - Training dataset with labels
- `test.csv` - Test dataset for predictions
- `submission.csv` - Generated predictions (created by the framework)

## Download Data

Download the datasets from the Kaggle Titanic competition:
https://www.kaggle.com/c/titanic/data

## File Descriptions

### train.csv
Contains the training data with the following columns:
- PassengerId
- Survived (target variable: 0 = No, 1 = Yes)
- Pclass (ticket class: 1 = 1st, 2 = 2nd, 3 = 3rd)
- Name
- Sex
- Age
- SibSp (# of siblings/spouses aboard)
- Parch (# of parents/children aboard)
- Ticket
- Fare
- Cabin
- Embarked (port of embarkation: C = Cherbourg, Q = Queenstown, S = Southampton)

### test.csv
Contains the test data with the same columns (except Survived).

### submission.csv
Generated by the Helios ML framework with predictions:
- PassengerId
- Survived (predicted values)
